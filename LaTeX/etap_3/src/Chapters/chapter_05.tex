\section{Podsumowanie}

W trzecim etapie projektu przeprowadzono optymalizację struktury sieci MLP
oraz parametrów procesu uczenia.

W szczególności przeanalizowano wpływ:
\begin{itemize}
    \item liczby warstw ukrytych i liczby neuronów,
    \item wartości współczynnika uczenia,
    \item stabilności wyników pomiędzy kolejnymi uruchomieniami.
\end{itemize}

Końcowa wersja sprawozdania powinna zawierać:
\begin{itemize}
    \item uzupełnione wykresy przebiegu funkcji kosztu,
    \item rzeczywiste tabele porównawcze dla wszystkich konfiguracji,
    \item interpretację wartości sensitivity i specificity,
    \item uzasadnienie wyboru najlepszej konfiguracji.
\end{itemize}
